Route Planner Map
-----------------
<h2>A route planner web application built with JavaScript, Leaflet, OpenStreetMap, and the OpenRouteService API.</h2>

Features
- Location search with autocomplete
- Driving, walking, and cycling routes
- Distance and travel time calculation
- Interactive map interface

Tech Stack
- `HTML`
- `CSS`
- `JavaScript`
- `Leaflet.js`
- `OpenStreetMap`
- `OpenRouteService API`

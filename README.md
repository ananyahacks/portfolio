# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML and CSS** to showcase my projects, skills, and contact information. This website serves as a digital space where I present my work, highlight my technical abilities, and provide an easy way for recruiters and collaborators to learn more about me.

---

## ✨ Features

* Responsive and clean UI design
* Structured sections for **About Me, Skills, Projects, and Contact**
* Simple and user-friendly navigation
* Lightweight and fast-loading static website
* Designed with modern layout and styling practices

---

## 🛠️ Technologies Used

* **HTML5** – for structuring the content
* **CSS3** – for styling and layout
* **Flexbox / modern CSS techniques** for responsive design
